About

Consultex is a student run nonprofit Fintech organisation that provides free and accurate return forecasts of index funds (BSE Sensex and NIFTY 50 index). We do this by analysing past data and market sentiment using Artificial Intelligence.

Our aim is to provide a general understanding of where the market is headed to help small investors avoid emotional trading.

Our code is completely open-source (Do consider contributing to the code if you can). Follow us on Instagram for more details.
